THE END OF THE TUNNEL — STORY ARC UPDATE

This build changes the former separate difficulty-level structure into one continuing five-part case arc.

The host still chooses a progressive case file for each session, and the cases still get harder, but they now continue the same larger story:

1. Case File 1: Peacock Alley — Training Detective
   Solves Dr. Malcolm Vire's murder and reveals the tunnel was opened recently.

2. Case File 2: The Missing Ledger — Rookie Detective
   Follows the first case into the hotel ledger and Page 47, proving old service codes are being used as modern dead drops.

3. Case File 3: The King Hotel Envelope — Junior Detective
   Reveals the King Hotel side is a live access route controlled through duplicate credentials.

4. Case File 4: The Heidelberg Escape — Detective
   Reveals the Heidelberg renovation route is being used to move forged artifacts and blackmail files.

5. Case File 5: The Door That Was Never Sealed — Senior Detective
   Reveals the mastermind and the full tunnel operation.

Render settings for this flat build:
Root Directory: leave blank
Build Command: npm install
Start Command: npm start
